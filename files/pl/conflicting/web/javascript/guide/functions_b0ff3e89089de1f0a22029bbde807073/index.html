Funkcje predefiniwane

JavaScript ma kilka predefiniowanych funkcji:

  • eval
  • isFinite
  • Funkcja isNaN
  • parseInt i parseFloat
  • Number i String
  • encodeURI, decodeURI, encodeURIComponent, i decodeURIComponent (wszystkie dostępne z Javascript 1.5 i późniejszym).
